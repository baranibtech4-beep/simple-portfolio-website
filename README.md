## Simple Portfolio Website

This is a simple personal portfolio website created using HTML and basic CSS. The project focuses on a clean design and fundamental web development concepts.

### Pages Included

- **Home Page**  
  Displays a brief introduction and overview of the portfolio.

- **Contact Page**  
  Provides basic contact information or a simple contact form for communication.

### Technologies Used

- HTML5  
- CSS3

### Purpose

This project was created to practice basic HTML structure, CSS styling, page linking, and simple layout design.

### Future Enhancements

- Add more pages like About, Skills, and Projects  
- Improve responsiveness  
- Add interactivity using JavaScript
